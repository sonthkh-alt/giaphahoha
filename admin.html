📖 Hướng dẫn sử dụng

  • Thêm/Sửa/Xóa thành viên trực tiếp trên giao diện.
  • Thay đổi được lưu tạm trong trình duyệt (localStorage).
  • Nhấn "Export JSON" để tải file family.json đã chỉnh sửa.
  • Upload file vào thư mục data/ trên repo GitHub để cập nhật website.
  • Nhấn "Import JSON" để tải dữ liệu từ file JSON khác.
  • Để xóa dữ liệu tạm (quay về bản gốc), xóa localStorage trong DevTools.
